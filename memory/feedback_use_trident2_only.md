Tout design Club Med est construit EXCLUSIVEMENT avec **Ody Trident Ψ 2.0** et **Main Trident Ψ 2.0**. C'est la règle la plus importante de ce projet, sans exception.

**Why:** C'est le DS officiel Club Med. L'utilisateur a dû le rappeler plusieurs fois — toute déviation est inacceptable.

**How to apply:**
1. Avant tout élément UI → chercher dans le DS via `mcp__figma__search_design_system` ou `mcp__figma__get_design_context`
   - Ody Trident Ψ 2.0 : `https://www.figma.com/design/EROcTR22vREhq7plcNLldI`
   - Main Trident Ψ 2.0 : `https://www.figma.com/design/tYBVCMeWwSsiQbdZrUOuoX`
2. Si trouvé → utiliser tel quel, jamais recréer
3. Si pas trouvé → demander à l'utilisateur avant de créer
4. Si un choix va à l'encontre du DS → demander l'avis et l'accord explicite de l'utilisateur avant de procéder
4. Jamais copier les styles d'un fichier client comme source de vérité DS
